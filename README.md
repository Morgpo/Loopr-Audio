## 🚀 Getting Started

### Installation

1. **Download** the latest release from the releases page
2. **Extract** the ZIP file to your preferred location
3. **Run** `LooprAudio.exe` to start the application

### First Launch

1. Click **"Browse"** to select your audio file (MP3, WAV, OGG, or M4A)
2. Click **"▶ Play"** to start looping
3. Adjust volume using the slider (0-100%)
4. Close the window to minimize to system tray

*That's it! Loopr Audio will remember your settings for next time.*

## 📋 How to Use

### Basic Controls

**🎵 Audio File Selection**
- Click **"Browse"** to choose your audio file
- Supported formats: MP3, WAV, OGG, M4A
- File name is displayed with hover tooltip for full path

**▶️ Playback Controls**
- **Play/Stop Button**: Start or stop audio playback
- **🔁 Loop Toggle**: Enable/disable continuous looping (enabled by default)
- **🚀 Start with Windows**: Automatically launch Loopr Audio when Windows starts

**🔊 Volume Control**
- Use the slider to adjust volume from 0-100%
- Or type directly in the percentage box
- Volume setting is saved automatically

### System Tray Features

When you close the main window, Loopr Audio minimizes to your system tray:

- **Left-click** the tray icon to restore the window
- **Right-click** for quick menu:
  - **Show**: Bring the main window back
  - **Play/Pause**: Control playback without opening the window
  - **Exit**: Close Loopr Audio completely

## ⚙️ Settings & Configuration

Loopr Audio automatically saves your preferences:

- **Last audio file**: Remembers your selected file
- **Volume level**: Keeps your preferred volume setting  
- **Loop setting**: Maintains loop on/off preference
- **Startup preference**: Whether to start with Windows

*All settings are stored locally in a `config.json` file in the application folder.*

## 🛟 Troubleshooting

**Audio Not Playing?**
- Ensure your audio file format is supported (MP3, WAV, OGG, M4A)
- Check that your system volume is not muted
- Try a different audio file to isolate the issue

**App Not Starting?**
- Make sure you have the complete application folder (don't run from inside a ZIP)
- Check that Windows isn't blocking the executable (right-click → Properties → Unblock)

**System Tray Issues?**
- If the tray icon doesn't appear, try restarting the application
- Check Windows notification area settings if the icon is hidden

**Performance Issues?**
- Loopr Audio uses minimal resources; if you experience issues, try closing other audio applications
- Large audio files (>100MB) may take longer to load initially
